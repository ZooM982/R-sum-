Un resumé sur le HTML & le CSS!

Rôle du HTML & du CSS

Le HTML gère et organise le contenu de la page.

Le HTML va donc créer la structure des pages tandis que le CSS va nous permettre de modifier l'apparence des contenus de la page. On va ainsi par exemple pouvoir définir la taille, la couleur ou l'alignement de certains contenus HTML et notamment en l'occurrence de certains textes dans notre page.

Questions/Réponses

C'est quoi le HTML et CSS ?

Le HTML est la structure technique de toute page de site web. Le CSS correspond au style, ce qui vous permet de montrer à l'utilisateur final le branding, les couleurs et les polices qui rendent votre identité de marque unique. Une page HTML, sans CSS, est fonctionnelle mais ne présente pas un contenu esthétique.

Quelle différence entre HTML et CSS ?

HTML et CSS sont les noms de différents types de code fréquemment utilisés dans le développement web. L'HTML permet l'affichage des informations de la page sur le navigateur, le CSS met en page son contenu

Comment utiliser HTML et CSS ?

Pour placer du code CSS dans une feuille HTML, il existe trois façons de procéder :

  1. Intégrer le code directement dans la balise HTML avec l'attribut style.
  2. Intégrer le code directement dans le fichier HTML avec l'attribut style.
  3. Créer un fichier CSS externe et relié à la feuille HTML via la balise link.

Quelle est l'utilité du langage CSS dans un ensemble HTML & CSS ?

Le CSS correspond à un langage informatique permettant de mettre en forme des pages web (HTML ou XML). Ce langage est donc composé des fameuses « feuilles de style en cascade » également appelées fichiers CSS (. css) et contient des éléments de codage.

Comment appeler un fichier CSS dans une page HTML ?

Il est possible d'écrire du code CSS dans l'entete d'un document HTML, dans la balise >head>. Il suffit pour cela de l'encadrer par une balise >style> ayant pour attribut type="text/css".

Quelques balises

Qui peuvent aider en HTML & CSS

Rôle des balises suivantes
  • <head> </head> Délimite l’en-tête de la page.
  • <title> </title> Définit le texte à afficher sur l’onglet de la page.
  • <body> </body> Délimite le corps de la page.
  • <image </> Ça n’existe pas (je connais <img /> pour mettre des images).
  • <meta charset="utf-8" </> Détermine l’encodage (comment les caractères spéciaux vont s’afficher) en utf-8.
  • <p> </p> Texte normal.
  • <br </> Retour à la ligne.
  • <h1> </h1> Titre de niveau 1 (titre).
  • <h2> </h2> Titre de niveau 2 (sous-titre).
  • <h3> </h3> Titre de niveau 3 (sous-sous-titre).
  • <em> </em> Mise en valeur normale.
  • <mark> </mark> Mise en valeur visuelle.
  • <strong> Mise en valeur forte.
  • <ul> </ul> Délimite une liste à puces non numéroté.
  • <li> </li> Ligne d’une liste à puces.
  • <ol> </ol> Délimite une liste à puce numéroté.
lien vers une autre page de son site:

<a href="(nom du fichier)"> Aller à cette page</a>

lien vers un autre site :

<a href="(url du site)"> Aller au site </a>

lien vers une autre partie de sa page html.

Dans une balise qui ouvre, on ajoute id="(nom)". Ensuite, dans pour faire un lien, on met : <a href="#(nom)">, et on atterrit à l’endroit de la page avec le id="(nom)".

lien vers une autre partie d’une autre page html.

<a href="(nom de l’autre page)#(nom de la balise)">

lien pour envoyer un email.

La balise qui ouvre le formulaire <form> doit avoir les attributs method="post" et action="(adresse mail)".

différence entre les formats d’images jpeg, png, gif

Le jpeg est beaucoup utilisé car le fichier n’est pas lourd (par rapport à d’autres formats, mais l’inconvénient c’est que ça détériore la qualité de l’image. Le png propose des images de très bonne qualité, et peut être rendu transparent. Le gif est limité à 256 couleurs, mais peut être animé.

insérer une image.
A quoi sert le mot alt

On insert une image avec la balise <img />, avec l’attribut src pour le nom de fichier de l’image. Le mot alt sert à afficher un texte quand l’image n’est pas trouvé.

A quoi servent les balises <figure> et <figcaption>

<figure> et <figcaption> permettent de lier une image, définit par <figure>, et la légende par <figcaption>.

indiquer l’utilisation d’une feuille de style css

<link rel="stylesheet" href="(nom du css)" />
